# **App Name**: ClubHub AI Landing Page

## Core Features:

- Hero Section: Displays headline, subheadline, and CTAs to showcase ClubHub AI's core value proposition.
- Demo Video Embedding: Embeds a responsive YouTube video demoing ClubHub AI's functionality, using the provided link: https://www.youtube.com/watch?v=siH9rm05vBM
- Workflow Feature Cards: Highlights key AI-powered workflows using a grid of feature cards, emphasizing benefits like automated announcement drafting and task management.
- Platform Relevance Section: Explains how ClubHub AI complements existing school communication platforms like ParentSquare, focusing on its workflow and orchestration capabilities.
- Contact Section with CTA: Encourages partnership inquiries with a clear contact section, including an email placeholder and an optional 'Request a meeting' button with a prefilled subject line.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey trustworthiness and a calm, professional feel.
- Background color: Light grey (#F5F7FA) for a clean, modern SaaS layout.
- Accent color: Muted purple (#B39DDB) for subtle highlights and call-to-action emphasis.
- Font pairing: 'Inter' (sans-serif) for headlines and body text.
- Single-scroll design with generous spacing and a max width of 900px for the video section, ensuring skimmability and mobile responsiveness.
- Subtle fade-in animations on scroll to enhance user engagement without being distracting.
